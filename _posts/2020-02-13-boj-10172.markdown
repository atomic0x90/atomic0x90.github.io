## BOJ - 10172 - 개

---

**설명**

`\` , `"` 를 문자 상수로 사용하는 법을 알아야 한다.

[\\ , " 를 문자 상수로 사용하는 법 알아보기][10]

---

**code**
```cpp
#include <iostream>

using namespace std;

int main()
{
        cout<<"|\\_/|"<<endl;
        cout<<"|q p|   /}"<<endl;
        cout<<"( 0 )\"\"\"\\"<<endl;
        cout<<"|\"^\"`    |"<<endl;
        cout<<"||_/=\\\\__|"<<endl;

        return 0;
}
```
