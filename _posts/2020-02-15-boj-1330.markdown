## BOJ - 1330 - 두 수 비교하기

---

**설명**

조건문을 사용하는 방법을 알아야 한다.

---

**code**
```cpp
#include <iostream>

using namespace std;

int main()
{
        double a,b;

        cin>>a>>b;

        if(a > b)
                cout<<">"<<endl;
        else if(a == b)
                cout<<"=="<<endl;
        else if(a < b)
                cout<<"<"<<endl;

        return 0;
}
```
