## BOJ - 2739 - 구구단

---

**설명**

반복문을 사용하는 방법을 알아야 한다.

[반복문(for) 사용하는 법 알아보기][10]

[반복문(while) 사용하는 법 알아보기][11]


---

**code**
```cpp
#include <iostream>

using namespace std;

int main()
{
        int a;

        cin>>a;

        for(int i = 1;i < 10;i++)
                cout<<a<<" * "<<i<<" = "<<a*i<<endl;

        return 0;
}
```
